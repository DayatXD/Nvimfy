## Features

- IDE like features
- Sweet Colorschemes
- File Navigation
- Tabs and Buffers
- Git Diffs
- Better Snippets
- AutoSave
- Syntax Highlighting
- AutoComplete
- Diagnostics
- Pretty StatusLine
- Terminal
- StartScreen
- Easy To Configure

## Install
```bash
[[ -f install.sh ]] && mv install.sh install.sh.old.$(date +'%Y-%m-%d_%R') ; wget https://raw.githubusercontent.com/AyeSpacey/Nvimfy/main/install.sh && chmod +x install.sh && ./install.sh
```
